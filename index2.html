¿Qué vas a aprender?

Este curso te guiará desde los fundamentos de la programación con Python hasta temas avanzados como POO, testing, automatización y buenas prácticas. Es 100% online y estructurado en clases semanales.

Programa: Introducción al Diseño Web

Este curso te enseña a crear sitios web desde cero. Aprenderás HTML, CSS, conceptos de estructura, diseño responsivo y buenas prácticas.

  • ¿Qué es la web? Cómo funciona un navegador.
  • Estructura básica de un documento HTML.
  • Estilos con CSS: colores, fuentes, márgenes.
  • Diseño adaptable (responsive design).
  • Prácticas con tu primer sitio web real.

Programa: Diseño Web Avanzado

Explorá técnicas modernas como Flexbox, CSS Grid, animaciones, diseño mobile-first, y uso de Tailwind. Ideal para dar el salto al diseño profesional.

  • CSS Grid y Flexbox en profundidad.
  • Animaciones con CSS y microinteracciones.
  • Optimización responsive avanzada.
  • Uso de Tailwind CSS y frameworks modernos.
  • Creación de landing pages profesionales.

Programa: Programación en Python

Desde variables y bucles hasta POO, funciones, testing, automatización y scripting real. Un curso completo y práctico en el lenguaje más versátil del mercado.

  • Variables, tipos de datos y estructuras de control.
  • Bucles for y while, condicionales.
  • Funciones y programación modular.
  • Programación Orientada a Objetos (POO).
  • Automatización de tareas y pequeños proyectos reales.
